CALCULADORA DE IP V4

1. Descripción:
   Aplicación web que calcula:
   - IP de red y broadcast
   - Rango de IPs útiles
   - Clase y tipo de IP
   - Representación binaria

2. Requisitos:
   - Python 3.9+
   - Rocky Linux 9
   - Puertos abiertos: 80/tcp

3. Uso:
   - Ingresar IP (ej: 192.168.1.100)
   - Ingresar máscara (ej: 24)
   - Click en "Calcular"

4. Ejemplos de prueba:
   - IP: 10.0.0.1, Máscara: 8
   - IP: 172.16.0.1, Máscara: 16
   - IP: 192.168.1.100, Máscara: 24

5. Características:
   - Manejo de redes /31 y /32
   - Identificación de IPs privadas
   - Validación de entrada
   - Interfaz responsive con Bootstrap
