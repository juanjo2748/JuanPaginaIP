Presentación del Proyecto

Volver a Calculadora

Esta aplicación fue desarrollada como una herramienta de apoyo para profesionales y estudiantes del área de Ingeniería de Sistemas, redes y telecomunicaciones. Su propósito principal es simplificar el análisis de direcciones IPv4, permitiendo calcular de manera rápida y precisa parámetros como:

  • Dirección de red
  • Dirección de broadcast
  • Rango de IPs disponibles
  • Cantidad de IPs útiles
  • Clase de la IP
  • Tipo de IP (Pública o Privada)
  • Representación binaria con separación de red y host

El sistema está construido usando Flask como framework web en Python, y se apoya en Bootstrap para una interfaz moderna, adaptable y profesional. El código utiliza módulos como ipaddress para garantizar cálculos precisos conforme a los estándares IPv4.

Su diseño está orientado a usabilidad, claridad de resultados y facilidad de integración en redes locales o servicios de laboratorio. Además, puede ser desplegado en servidores con sistemas como Rocky Linux 9, configurado para ejecutarse en el puerto 80, ideal para entornos empresariales o educativos.

Este proyecto refleja el compromiso con la excelencia técnica y la aplicación práctica del conocimiento en Ingeniería de Sistemas.
